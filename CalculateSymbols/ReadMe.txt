
Write a function that takes an input string with characters. On the output we get the number of different characters that occur on the line.
For example:
entrance: "aaabcadac";
output: "a": 5;
       "b": 1;
       "c": 2;
       "d": 1;

Написать функцию, которая принимает на вход строку с символами. На выходе получаем количество разных символов, которые встречабтся в строке.
Например:
вход: "aaabcadac";
выход: "a" : 5;
       "b" : 1;
       "c" : 2;
       "d" : 1;
