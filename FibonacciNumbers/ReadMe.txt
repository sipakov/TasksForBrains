There are a number of 1,1,2,3 ... Fibonacci series.
                      1,2,3,4     Numbers of Fibonacci numbers. Natural series.
Write a function that takes two arguments. If the first argument (the number of the Fibonacci number) on the input, then displays the corresponding Fibonacci number.
If the input is the second argument (a number from the Fibonacci series or any other number), it outputs the corresponding Fibonacci sequence number
if it matches the entered one or displays the nearest larger Fibonacci number if the number does not enter the Fibonacci sequence.
Estimate the speed of the speed of algorithms.
